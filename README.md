# AI Automation Webinar

A modern, responsive web application for the AI Automation Webinar, designed to showcase course details, trainer information, and facilitate seat reservations. Built with [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/), and [Tailwind CSS](https://tailwindcss.com/).

## ðŸ“¦ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MudassarHakim/ai_automation_webinar.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ai_automation_webinar
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```
